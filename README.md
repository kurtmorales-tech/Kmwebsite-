# Kmwebdesign Agency Website

This is the official website for Kmwebdesign, a digital agency founded by Kurt Morales.
Built with Next.js, Tailwind CSS, and Radix UI.

## Features

- **Responsive Design**: Mobile-first approach.
- **Top Performance**: Optimized for Core Web Vitals.
- **SEO Ready**: Static metadata, robots.txt, and sitemap.xml.
- **Accessibility**: ARIA labels and semantic HTML.

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Deployment

This project is optimized for Vercel.

1. Push to GitHub.
2. Import project in Vercel.
3. Deploy!

## Customization

Content is managed in `src/data/content.ts`. Update this file to change text, links, and branding.
